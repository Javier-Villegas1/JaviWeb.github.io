HTML Tags

In this section, we'll explore various HTML tags and their use.

Resumen de Tags HTML

  • <html>: Define el documento HTML.
  • <head>: Contiene metadatos (como título, enlaces a CSS).
  • <body>: Contiene el contenido visible de la página.
  • <h1> - <h6>: Encabezados, de mayor a menor importancia.
  • <p>: Define un párrafo de texto.
  • <a>: Crea enlaces (hipervínculos).
  • <img>: Inserta imágenes.
  • <div>: Contenedor genérico para agrupar elementos.
  • <span>: Contenedor en línea para aplicar estilos.
  • <ul> / <ol>: Listas desordenadas u ordenadas.
  • <li>: Elemento de lista.
  • <form>: Define un formulario.
  • <input>: Campo de entrada de datos en formularios.
  • <table>: Crea tablas con filas y columnas.
  • <footer>: Pie de página, generalmente con información adicional.

This is a big resume of the principal tags of HTML and for what their are used.
