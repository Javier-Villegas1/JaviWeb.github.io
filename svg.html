SVG Tags

In this section, we'll explore various SVG formats, their use and examples.

Resumen de SVG

  • color: Define el color del texto.
    Ejemplo: color: blue;
  • background-color: Establece el color de fondo de un elemento.
    Ejemplo: background-color: yellow;
  • font-size: Controla el tamaño de la fuente.
    Ejemplo: font-size: 16px;
  • font-family: Especifica la fuente del texto.
    Ejemplo: font-family: Arial, sans-serif;
  • margin: Define el espacio exterior alrededor de un elemento.
    Ejemplo: margin: 20px;
  • padding: Define el espacio interior entre el contenido y el borde de un elemento.
    Ejemplo: padding: 10px;
  • border: Establece el borde de un elemento.
    Ejemplo: border: 1px solid black;
  • width / height: Controla el tamaño de un elemento.
    Ejemplo: width: 100px; height: 50px;
  • display: Controla cómo se muestra un elemento (por ejemplo, en bloque, en línea).
    Ejemplo: display: block;
  • position: Controla la posición de un elemento (relativa, absoluta, fija, etc.).
    Ejemplo: position: absolute;
  • flex: Define un contenedor flexible para organizar elementos.
    Ejemplo: display: flex; justify-content: center;
  • grid: Crea un diseño de cuadrícula para organizar elementos.
    Ejemplo: display: grid; grid-template-columns: 1fr 1fr;
  • text-align: Alinea el texto dentro de un contenedor.
    Ejemplo: text-align: center;
  • box-shadow: Agrega una sombra a un elemento.
    Ejemplo: box-shadow: 2px 2px 5px gray;
  • opacity: Define la transparencia de un elemento.
    Ejemplo: opacity: 0.5;

This is a big resume of how you can use SVG in a web and for what you can use it.
